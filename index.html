My Project Work

Shown below is some of my project work, and a bit about what it means to me and why it's relevant.

Cal Eats

This is a full-stack web application that allows students to post ratings and reviews of Berkeley dining hall food. (More informed dining decisions can then be made!)

I worked with a team of 5 people to build this. Technologies used: MongoDB, JavaScript (React), Python (Flask)

Github Link

Hangman game

This is a game of Hangman I built in my first semester of college. Having recently written my first lines of code, I wanted to go further and built my variation of this game I loved as a kid.

I implemented this in Python and used Tkinter for the graphical user interface.

Github Link

Ocean trash dashboard

This is a dashboard made for an organization called Pacifica Beach Coalition to draw attention to the issue of ocean trash.

I worked with a team of 8 people to build this. Technologies used: Python (Plotly Dash library), Jupyter Notebook, HTML, CSS and Heroku.
